Important FAQ

What is the Document Object Model (DOM)
DOM is a programming interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.
How do you select an element from the DOM?
We can select an element from DOM by using document.getElementById() and document.querySelector()
What is event delegation in the context of DOM and why it is useful?
Event delegation is a technique in the DOM where you attach a single event listener to a parent element instead of multiple listeners to individual child elements. It is useful for performance, dynamic elements and simpler code .
How do you manipulate an elements attributes and styles using the DOM?
To manipulate an element's attributes and styles using the DOM, we can use methods like getAttribute(), setAttribute(), and removeAttribute() to modify HTML attributes such as id, class, or custom data-* attributes. For styling, the element.style property allows direct inline CSS changes (e.g., element.style.color = 'red'), while classList methods (add(), remove(), toggle()) help manage CSS classes efficiently.
